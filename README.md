## 📖 Le Contexte

Un vent de suspicion souffle sur les couloirs d'Epitech Montpellier... 🌬️

Un membre de l'équipe pédagogique, un certain **Bob Morane**, attire l'attention. Plusieurs indices laissent penser qu'il ne serait pas un simple pédago, mais un **agent secret infiltré** envoyé par la mystérieuse société concurrente : **Ekitech**.

Sa mission ? Espionner nos méthodes de pointe. Votre mission ? **Le démasquer.**

L'enquête commence autour de son blog personnel. Ce qui semble être un site banal cache en réalité des secrets bien gardés. Qu'allons-nous découvrir sur ce fameux Bob ?

---

## 🎯 Objectif de l'Atelier

Votre but est de rassembler des preuves irréfutables de ses activités d'espionnage.

* **Le Défi :** Résoudre une série de challenges techniques (OSINT, Web, Cryptographie, etc.).
* **La Récompense :** Chaque challenge validé vous octroie une **pièce du puzzle** (une preuve).
* **Le Quota :** Il y a un total de **14 preuves** à collecter pour boucler le dossier.

---

## 🛠️ Missions & Preuves

L'étudiant doit explorer l'écosystème numérique de Bob Morane pour extraire les données sensibles.

| Preuves Collectées | Statut de l'enquête |
| :--- | :--- |
| **0 - 3** | Amateur : Bob se rit de vous. |
| **4 - 7** | Détective : Vous commencez à l'inquiéter. |
| **8 - 9** | Agent Spécial : La vérité est proche. |
| **10** | **Expert Interpol : Dossier classé, Ekitech est démasqué !** |
| **11 et +** | **Expert International : Incroyable.** |

---

## 🏆 Classement

La compétition est rude ! Un classement sera établi pour déterminer qui sera le meilleur enquêteur du Coding Club. Celui qui récoltera les **10 preuves** le plus rapidement remportera la palme du meilleur agent.

> 💡 **Conseil :** Ne vous fiez pas aux apparences. Le code source d'une page cache souvent plus de secrets qu'un long discours.

---

## 🚀 Comment enquêter ?

1.  **Analysez le blog** : Rendez-vous sur le site de Bob : URL.
2.  **Outils recommandés** :
    * Inspecteur d'élément (F12 est votre meilleur ami).
    * Outils de décodage (Base64, CyberChef, EXIF Tools).
    * Votre cerveau et un peu d'intuition !
3. **Lisez attentivement le blog**, beaucoup de preuves sont juste données.

---

## Quand vous trouvez une preuve :

- Si elle mentionne **EKITECH** ou expose les activités de Bob, **prennez une photo** pour nous la montrer à la fin.
